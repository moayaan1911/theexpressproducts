# Products API file

## RUN ON YOUR LOCAL MACHINE
- clone this repo by typing ```git clone ```
- upload the sample data set file on mongoose OR upload your own
- create .env with same credentials as mentioned in .env.example
- Run the endpoints mentioned in server.js


## Endpoints
- Get all products OR Delete all products
  ```/products```
- Get particular product OR Update particular product OR Delete particular product
  ```/products/:id```

( Use POST,PUT,GET,DELETE accordingly )
